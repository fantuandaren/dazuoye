# 森林冰火人

## 项目简介
本项目为 C++/Qt 平台开发的《森林冰火人》双人协作闯关小游戏。游戏融合了平台跳跃与益智解谜玩法，主打双角色分工协作，共同通关。

## 角色设定

- **火娃 (Fireboy)：**
  - 可在熔岩、火焰上行走
  - 遇水/绿色毒液死亡
  - 收集红宝石，从红门离开

- **水娃 (Watergirl)：**
  - 可在水池中穿行
  - 遇火/熔岩死亡
  - 收集蓝宝石，从蓝门离开

- **共同规则：**
  - 绿色毒液及泥沼对两者都致命
  - 白色/灰色宝石可由任意角色收集

## 核心玩法与机关

- 元素机关：火娃/水娃需各自通过相应的特殊关卡
- 压力开关：需两角色配合激活平台或门
- 移动平台与吊梯：考验配合与操作时机
- 道具运用：合作推动石块、铺路等
- 支持双人协作和单人切换模式

## 运行方法
1. 安装 Qt（建议 Qt 4.x/5.x）及 VS2010
2. 用 Qt Creator 或 VS2010 打开 `.pro` 工程文件
3. 编译并运行项目
